# Kylian — Personal Website

A bilingual editorial portfolio about product judgment, agent evaluation, geospatial workflows, creative tools, and F1.

## Local development

```sh
npm install
npm run dev
```

## Verification

```sh
npm run check
```

The verification command runs Astro diagnostics, generates the static site, checks internal links, and scans source and output files for private identifiers or local paths.

## Deployment

Pushes to `main` deploy automatically to GitHub Pages through the workflow in `.github/workflows/deploy.yml`.
